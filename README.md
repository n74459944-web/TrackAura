# TrackAura: Global Item Value Tracker

TrackAura.com—Your one-stop site to monitor every item's worth worldwide. Features:
- **Historical Pricing Charts**: Line graphs of value over time (e.g., crypto, gadgets, collectibles).
- **Item Pages**: Full specs, related info, and deal alerts.
- **Category Galleries**: Visual grids with pics for easy browsing (Electronics, Fashion, etc.).

## Tech Stack
- Frontend: Next.js 14 (React, TypeScript, Tailwind)
- Viz: Chart.js for price trends
- Backend: Prisma + Supabase (TBD)
- Deployment: Vercel

## Local Setup (Windows)
1. `git clone https://github.com/n74459944-web/TrackAura.git`
2. `cd TrackAura`
3. `npm install`
4. `npm run dev` → Open http://localhost:3000
